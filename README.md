# 아이들의 낙서장

https://twitter.com/popnmusic10 에서 만드는 팝픈뮤직 서열표의 메달 정보를 자동으로 기입해주는 스크립트의 저장소입니다.

저장소 이름은 사사키 히로후미의 [子供の落書き帳](https://remywiki.com/Kodomo_no_rakugaki_chou)에서 따왔습니다.
